# Blockchain em Java
Uma blockchain simples desenvolvida em Java.

## Funcionalidades
- Sistema básico de mineração utilizando **Proof of Work**
- Criação de wallets simples
- Envio de transações assinadas na blockchain
  
## Tecnologias
- Java
- Eclipse IDE
- Gson
- Bouncy Castle

## Como executar
1. Clone o repositório:
git clone https://github.com/mariliamarquess/blockchain-java

2. Abra no Eclipse

3. Adicione as dependências externas:
- Gson
- Bouncy Castle

>> Baixe as bibliotecas (.jar) e adicione ao projeto:
Right click no projeto → Build Path → Add External JARs

4. Execute a classe principal
